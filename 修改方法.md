2. 插件修改

   先下载插件 https://github.com/obgnail/typora_plugin 使用 vscode 打开 window.html

   如果不想下载使用我配置好的也可以。文件夹在： resources\plugin 在发布界面中我会打包好。

   ```bash
   	<!--  搜索 frame 找到这句 -->
   	<script src="./appsrc/window/frame.js" defer="defer"></script>
   	<!--  添加下面这条代码 -->
   	<script src="./plugin/index.js" defer="defer"></script>
   ```
